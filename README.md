# The Beauchamp Family Line

A personal family-history project covering 31 generations and ~950 years — from Hugh de Beauchamp of Bedford (Domesday tenant-in-chief, d. ~1101) to the modern Gaddis–Carlson family.

After Lorraine Shoemaker's original compilation, expanded with primary-source research.

## Views

- **[Panorama](beauchamp_panorama.html)** — pan/zoom canvas with every person; click any name for full notes and sources.
- **[Reference](beauchamp_reference.html)** — twelve-section narrative document covering each branch in turn.
- **[Summary](beauchamp_summary.html)** — short overview.
- **[Maternal line](beauchamp_maternal_line.html)** — Carlson / Fehlberg / Pschigoda / Norwood detail.

## Privacy

This site is **unlisted** — search engines are blocked via `robots.txt` and `noindex` meta tags. Anyone with a direct link can view it.

## Source

The site is fully self-contained static HTML; no server-side code, no external image dependencies (every portrait, photo, and document scan is embedded).
